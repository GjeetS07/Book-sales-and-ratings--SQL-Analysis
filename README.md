Book Sales and Ratings Analysis (SQL Project)

Project Overview
This project analyzes book sales and rating data using SQL to identify key trends in publishing performance, customer preferences, and revenue generation. It combines multiple analytical techniques such as joins, common table expressions (CTEs), subqueries, and window functions to extract meaningful insights.

Tools and Technologies
* SQL (MySQL)
* Data Analysis
* Analytical Problem Solving

Dataset Description
The dataset consists of two tables:

Books Table
Book details such as name, author, publisher, genre, and publishing year

Ratings Table
Sales and performance metrics including:
  * Gross sales
  * Publisher revenue
  * Book ratings and counts
  * Units sold and pricing


Sales and Revenue Analysis
* Top-selling books based on gross sales
* Publishers generating highest total revenue
* Profit margin analysis for each book
* Total sales trends by publishing year

Customer Preference Analysis
* Average rating by genre
* Most rated books in English language
* High-rated but low-selling books (hidden opportunities)

Author Performance Analysis
* Authors with multiple books and their average ratings
* Authors whose personal rating exceeds book ratings

Targeted Insights
* High-rated books published in a specific year
* Identification of underperforming yet highly rated books

Key Insights
* Certain genres consistently achieve higher average ratings
* A small number of publishers dominate total revenue generation
* Some books have high ratings but low sales, indicating missed market opportunities
* Authors with multiple books tend to maintain consistent performance levels
* Sales trends vary significantly across publishing years
